# Darshan Bhandigare Portfolio

This project is now a React + Vite version of the original static portfolio.

## Tech Stack

- React
- Vite
- CSS

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## What Changed

- Converted the page into React components
- Kept the original portfolio content and styling
- Moved browser effects like the particle canvas, custom cursor, and reveal animations into React effects

## Build Status

The project builds successfully with `npm run build`.
